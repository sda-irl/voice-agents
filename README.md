 ## Voice Service Desk Agent (ElevenLabs)

This project demonstrates a voice-based IT Service Desk Assistant built using ElevenLabs. The agent is designed to simulate real-world service desk interactions and support common technical scenarios.

---

## Project Overview

The agent uses structured knowledge base content and defined conversation branches to respond to user requests. It can handle general questions, troubleshooting, service requests, and frequently asked questions.

---

## Knowledge Base

The agent is trained on four internal-style documents:

- IT Service Desk Overview
- Account Access and Security Policy
- Common Technical Issues and Troubleshooting
- Service Requests and Onboarding

These sources provide consistent and related information to support natural conversation flow.

---

## Conversation Branches

The agent is designed with four main conversation paths:

1. General Information  
2. Troubleshooting  
3. Service Requests  
4. FAQ  

Each branch supports a different user intent and helps guide the conversation effectively.

---

## Example Use Cases

- Resetting passwords or unlocking accounts
- Troubleshooting login or email issues
- Requesting new equipment or access
- Answering policy or process questions

## Reflection Questions

### Branch Design Challenge
The most challenging part of designing the conversation branches was ensuring overlap between user intents did not create confusion. Many user requests could fall into multiple categories, which required careful structuring of responses to keep conversations consistent.

### Knowledge Base Impact
The knowledge base significantly improved response quality. Content that included clear steps and structured explanations worked better than general descriptions because the agent could guide users more effectively.

### Real-World Application
This voice agent could be used in higher education IT departments to assist students, faculty, and staff with common technical issues. It could reduce ticket volume, provide after-hours support, and improve response times for routine requests.
